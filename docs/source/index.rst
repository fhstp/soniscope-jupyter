
soniscope_jupyter
=====================================

Version: |release|

jupyter notebook widget with a scatter plot and an interactive lens to enable interactive sonification


Quickstart
----------

To get started with soniscope_jupyter, install with pip::

    pip install soniscope_jupyter

or with conda::

    conda install soniscope_jupyter
