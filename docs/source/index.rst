
sonivis_lens_widget
=====================================

Version: |release|

jupyter notebook widget with a scatter plot and an interactive lens


Quickstart
----------

To get started with sonivis_lens_widget, install with pip::

    pip install sonivis_lens_widget

or with conda::

    conda install sonivis_lens_widget
